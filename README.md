### Markdown

Markdown是一种轻量级和易于使用的语法，用于设计您的写作风格。它包括约定如下

```markdown
Syntax highlighted code block

# Header 1
## Header 2
### Header 3

- Bulleted
- List

1. Numbered
2. List

**Bold** and _Italic_ and `Code` text

[Link](url) and ![Image](src)
```

 [GitHub Flavored Markdown](https://guides.github.com/features/mastering-markdown/).
